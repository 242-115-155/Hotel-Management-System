# 🏨 NS Grand Hotel Management System

A modern Hotel Management System built with **PHP**, **MySQL**, **HTML5**, **CSS3**, **Bootstrap 5**, and **JavaScript**. The system allows customers to browse rooms, make bookings, and manage reservations, while administrators can efficiently manage rooms and booking requests through a dedicated dashboard.

---

## 📌 Features

### 👤 Customer
- View available rooms
- Room image gallery
- Book a room
- Cancel booking
- Booking history
- Online payment page
- Responsive user interface

### 🔑 Admin
- Secure admin login
- Add new rooms
- Modify room details
- Remove rooms
- View room status
- Confirm booking requests
- View confirmed bookings
- Booking history management

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Hotel-Management-System.git
```

2. Move the project folder to:

```
xampp/htdocs/
```

3. Start

- Apache
- MySQL

from the XAMPP Control Panel.

4. Import the database

- Open **phpMyAdmin**
- Create a new database
- Import the **database.sql** file

5. Open your browser

```
http://localhost/Hotel-Management-System/
```
